Phrase list / pronunciations

When the TTS engine encounters Print, it says Phonetic instead. Use plain re-spellings for "sub" type (default) or IPA/SAPI for power users. In sub mode you can mark a syllable's prosody inline: ALL CAPS = stressed (e.g. Yah-HOH-wah), [brackets] = drawn out / slowed (e.g. Yah-[hoh]-wah), {braces} = clipped / quickened (e.g. Yah-{hoh}-wah). Markers stack: [SHA]-baht is both stressed and drawn out.
