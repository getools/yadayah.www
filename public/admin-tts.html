Phrase list / pronunciations

When the TTS engine encounters a row's Text string in the source it substitutes the IPA pronunciation instead, wrapped in an Azure <phoneme alphabet="ipa"> tag. Each column controls one piece of how that substitution behaves:

  • Active — checkbox at the left toggles the rule on/off without deleting the row.
  • Text — the source-text needle. Apostrophe-equivalent characters (straight, curly, half-rings) are interchangeable when the needle contains one; a trailing possessive 's is matched separately and the phonetic gets a z appended. B / I / Aa after the input restrict matching to bold runs, italic runs, or case-sensitive — multiple restrictions combine.
  • # — how many times this Text appears across all parsed books. Sortable.
  • IPA — phonetic spelling using IPA symbols. Click the ⓘ for a popover of common IPA characters with example words. Stress marks: ˈ primary, ˌ secondary; length: ː.
  • SAPI — reference-only spelling using legacy SAPI phoneme codes (AH1 HH OW0). Azure neural voices reject SAPI, so this column is preserved for cross-checking against older tools; not used by synthesis.
  • ▶ — auditions this rule using the category voice selected in the picker above the table, with that category's saved style, rate, pitch, and volume.
  • Voice — optional per-tune voice override. Setting one on a previously-voiceless row creates a clone (the original stays as the voice-NULL catch-all); only one voice-specific clone stays active per Text, others become inactive automatically.
  • Sort — manual priority for breaking ties between overlapping rules. Higher value wins; blank/0 means use defaults (Text length, then row order).
  • Note — pencil icon opens a popover for free-form notes (intent, edge cases, etc.). Amber when a note is set.
  • × — delete the row.

Tunes apply before pause rules during synthesis, so the matched text consumes its half-ring / apostrophe characters before the pause filter ever sees them.

Pause rules

When the engine sees the Character(s) string in the text, insert a pause of the given length. Use surrounding spaces (e.g.  - ) to anchor a match — without spaces every occurrence of the character pauses, including inside words. Set ms = 0 to suppress Azure's natural prosodic pause at that spot (handy for removing the small pause inside hyphenated compounds like "first-person").

Font filter

Source documents tag each text run with a font. When TTS encounters a font marked Skip, that text is silently dropped. Pause on switch inserts a <break> when the engine transitions into that font (handy for skipped scripts so the listener perceives a beat where the Hebrew was).
