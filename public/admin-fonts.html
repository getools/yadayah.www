Fonts

Central font registry. Used by every text editor on the site (TinyMCE on Pages, Ask, etc.).
Group: empty = main alphabetical list at the top. Number (1, 2, ...) = grouped below the main list, with a divider before each group.
Sort: tie-breaker within a group; lower numbers first.
Sample / Glyph: short text shown in the font on the dropdown (e.g. HWHY for Paleo-Hebrew). Leave blank to show the font's display name in its own font (standard preview behavior).
CSS family: the font stack applied when this entry is picked. Wrap multi-word names in quotes. Examples: YadaTowrah-Times,serif, "Maven Pro",sans-serif.
