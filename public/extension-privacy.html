Privacy Policy — Yada Yah Transcription Helper

Last updated: 2026-05-01

What this extension does

The Yada Yah Transcription Helper is a Chromium browser extension that lets administrators of yadayah.com send YouTube video audio (or YouTube caption tracks) to the yadayah.com server for transcription. It is designed for use by site administrators only.

What data the extension accesses

  • YouTube watch pages — when you visit youtube.com/watch?v=..., a content script reads the page's ytInitialPlayerResponse object to find caption track URLs.
  • googlevideo.com network requests — the extension's background worker observes audio-stream requests YouTube makes during playback so it can later download the same audio file.
  • The active tab's URL — used to identify the YouTube video ID when capturing audio.
  • Your yadayah.com session cookie — sent automatically by the browser when uploading audio to https://yadayah.com/api/admin-transcript-upload.php. The extension does not read or store this cookie itself.
  • Settings you enter — server URL and last-used feed item key, stored locally via chrome.storage.local.

What data the extension transmits

  • Audio data — when you click "Send Audio", the extension downloads the YouTube audio stream (using your IP and session, the same way YouTube's player would) and uploads it to https://yadayah.com/api/admin-transcript-upload.php for transcription.
  • Caption text — when you click "Send Captions", the extension fetches YouTube's caption track and uploads it to the same endpoint.
  • The destination is always yadayah.com — there are no third-party servers and no analytics.

What data the extension does NOT collect

  • No browsing history, page content, form input, financial information, personally identifiable information, health information, or authentication credentials.
  • No tracking, no telemetry, no analytics — first or third party.
  • The extension does not transmit any data to any server other than yadayah.com.

Data retention

Local settings (server URL, last-used item key) stay in your browser's local storage until you uninstall the extension or clear browser data. Captured audio URLs are kept only in chrome.storage.session (cleared when Chrome restarts) and are removed automatically when the corresponding tab closes.

Audio or caption files you upload to yadayah.com are stored temporarily on the yadayah.com server in /tmp/transcript_uploads/ until processed by the transcription worker, then removed. Transcribed text is stored in the yadayah.com database under the administrator's account.

Permissions and why each is needed

  • webRequest — observe googlevideo.com audio-stream URLs during YouTube playback so we can fetch the audio later.
  • storage — remember the server URL and last-used feed item key locally.
  • tabs — identify which tab is the active YouTube watch page.
  • scripting — inject a small script into YouTube watch pages to read ytInitialPlayerResponse for caption-track URLs.
  • notifications — show a desktop notification when an audio upload completes.
  • Host permission for *.youtube.com / *.googlevideo.com — read caption tracks and download audio.
  • Host permission for yadayah.com — upload captured audio/captions to the destination server.
